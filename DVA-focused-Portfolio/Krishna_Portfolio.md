## Featured Projects

### 1. NASA Planetary Defense Analytics Dashboard
*Data Visualization & Analytics (DVA) Capstone Project*

**Link:** [Planetary Defense Analytics Dashboard](https://github.com/CosmicMagnetar/SectionC_Group09_ExoPlanets)

**Project Context:**
NASA tracks over 41,000 Near-Earth Asteroids (NEAs) and monitors tens of thousands of close approach events. The objective of this project was to parse, clean, and map highly complex, abbreviated orbital mechanics datasets from JPL into intuitive, decision-ready visual models that can effectively identify Potentially Hazardous Asteroids (PHAs).

**Key Responsibilities & Contributions:**
- **Data Engineering & ETL Pipelines:** Engineered robust Python-based pipelines utilizing Pandas to clean and process over 68,000 asteroid records across two primary datasets. Addressed critical data quality issues by handling missing values and normalizing complex orbital terminology.
- **Risk Categorization & Logic:** Designed and implemented custom algorithmic logic to categorize asteroids into actionable risk tiers (Critical, High, Moderate, Low). This categorization was based on the Minimum Orbit Intersection Distance (MOID) threshold of 0.05 AU and absolute magnitude limits.
- **Visual Analytics Architecture:** Contributed to the development of interactive Tableau dashboards. These visual interfaces were designed to track future close approaches up to 2035, enabling users to filter by orbit class, speed category, and hazard level, thereby transforming static data into dynamic intelligence.
- **Technical Stack:** Python (Pandas, SciPy), Tableau, Jupyter Notebooks, GitHub

### 2. Aadhaar Dataset Analysis & Preprocessing
*UIDAI Data Hackathon Submission*

**Link:** [Aadhaar Dataset Analysis & Preprocessing](https://colab.research.google.com/drive/1xZMVYfXJ9rNGgilZQ2J-ykUjW2t_lvXF)

**Project Context:**
Working with massive, real-world demographic datasets requires rigorous preprocessing to ensure data integrity before any predictive modeling can occur. This project involved extensive data cleaning and analysis of a large-scale demographic dataset for the UIDAI Data Hackathon.

**Key Responsibilities & Contributions:**
- **Advanced Data Cleaning:** Built scalable data imputation and normalization techniques to resolve anomalies, handle malformed records, and manage missing values across millions of data points without compromising dataset validity.
- **Exploratory Data Analysis (EDA):** Conducted deep-dive statistical analysis to surface underlying demographic patterns and distributions. Created insightful visualizations to guide feature selection and inform the subsequent machine learning modeling phases.
- **Performance Optimization:** Structured and optimized the Python processing scripts within Google Colab environments. Ensured memory-efficient execution and rapid processing times when handling high-volume CSV operations.
- **Technical Stack:** Python, Pandas, Matplotlib, Google Colab
